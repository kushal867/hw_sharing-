# Homework Sharing Platform

A Django-based web application for managing and sharing homework, submissions, and profiles for teachers and students.

## Features

- **User Management:** Register, login, and manage profiles.
- **Profile System:** Teachers and students can have custom profiles with roles, bio, and profile pictures.
- **Subjects & Homework:** Teachers can create subjects and assign homework.
- **Submissions:** Students can submit homework and track approval status.
- **Comments:** Users can comment on homework assignments for discussion.
- **Responsive UI:** Clean, mobile-friendly interface using Bootstrap.
